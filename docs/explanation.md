# サービス名

Itineos (アイティネオス)

Itinerary（旅程）+ Neos（新しい）。

「新しい旅の形」を提案する先進的なイメージ。

# 説明

インバウンド向け「スマート観光ルート・コンシェルジュ」

訪日外国人観光客が「行きたい場所」と「現在の気分」を入力するだけで、移動手段・天候・混雑状況を考慮した最短・最適な観光ルートを自動生成するサービスです。

組み合わせる API:

- Google Maps Platform (Routes API): 徒歩・電車・タクシーを組み合わせた正確な移動時間とルートを算出します。
- OpenWeatherMap API: 指定した場所のピンポイントな天気予報を取得し、雨天時の代替ルート提示に活用します。
- Gemini API (Google AI Studio): ユーザーの曖昧な要望（例：「静かな場所で抹茶を飲みたい」）を解析し、最適なスポットを選定します。

収益化モデル: 地域店舗からの広告掲載料（ルート上の店舗紹介）や、ホテル・タクシー予約の送客手数料。
